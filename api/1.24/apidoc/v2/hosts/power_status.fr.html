GET /api/hosts/:id/power
Fetch the status of whether the host is powered on or not. Supported hosts are VMs and physical hosts with BMCs.

Exemples

GET /api/hosts/43/power
200
{
  "id": 43,
  "state": "on",
  "title": "On"
}
