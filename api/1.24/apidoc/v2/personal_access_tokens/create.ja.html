POST /api/users/:user_id/personal_access_tokens
ユーザーのパーソナルアクセストークンを作成

例

POST /api/users/988725681/personal_access_tokens
{
  "personal_access_token": {
    "name": "foreman@example.com",
    "expires_at": "2019-12-05T08:45:43Z"
  }
}
201
{
  "token_value": "SK-PrjW98OTY1h8QTY9hUA",
  "created_at": "2019-11-07 08:45:43 UTC",
  "updated_at": "2019-11-07 08:45:43 UTC",
  "id": 5,
  "name": "foreman@example.com",
  "expires_at": "2019-12-05 08:45:43 UTC",
  "active?": true,
  "user_id": 988725681,
  "last_used_at": null
}

Validations:

  • Must be a Integer
user_id
必須

ユーザーの ID

Validations:

  • Must be a String

必須

Validations:

  • Must be a Hash
