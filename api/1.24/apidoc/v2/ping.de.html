>>>

GET /api/ping
Shows status of Foreman system and it's subcomponents

This service is available for unauthenticated users

Beispiele

GET /api/ping
200
{
  "results": {
    "foreman": {
      "database": true
    }
  }
}

Parameter

Parametername Beschreibung
location_id
optional

Set the current location context for the request

Validations:

  • Must be a Integer
organization_id
optional

Set the current organization context for the request

Validations:

  • Must be a Integer
>>>

GET /api/statuses
Shows status and version information of Foreman system and it's subcomponents

This service is only available for authenticated users

Beispiele

GET /api/statuses
200
{
  "results": {
    "foreman": {
      "version": "1.20.0",
      "api": {
        "version": "v2"
      }
    }
  }
}

Parameter

Parametername Beschreibung
location_id
optional

Set the current location context for the request

Validations:

  • Must be a Integer
organization_id
optional

Set the current organization context for the request

Validations:

  • Must be a Integer
