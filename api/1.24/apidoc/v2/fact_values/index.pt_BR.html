GET /api/fact_values
Listar todos os valores de fatos

GET /api/hosts/:host_id/facts
Listar todos os valores de fatos de um host específico

Exemplos

GET /api/fact_values
200
{
  "total": 0,
  "subtotal": 1,
  "page": 1,
  "per_page": 20,
  "search": "host.hostgroup = hostgroup58",
  "sort": {
    "by": null,
    "order": null
  },
  "results": {
    "host621": {
      "kernelversion": "2.6.9"
    }
  }
}
