GET /api/report_templates
レポートテンプレートをすべて表示

GET /api/locations/:location_id/report_templates
ロケーションごとのレポートテンプレートをすべて表示

GET /api/organizations/:organization_id/report_templates
組織ごとのレポートテンプレートをすべて表示

例

GET /api/report_templates
200

  },
  "results": [
    {
      "description": null,

}

任意

Validations:

  • Must be a Integer

任意

結果のフィルター

Validations:

  • Must be a String

任意

Validations:

任意

  • Must be a number.

任意

Validations:
