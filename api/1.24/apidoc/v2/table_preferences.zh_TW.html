POST /api/users/:user_id/table_preferences
Creates a table preference for a given table

示例

POST /api/users/135138680/table_preferences
{
  "name": "test_subscriptions_resource",
  "columns": [
    "1",
    "2",
    "3"
  ],
  "table_preference": {
    "name": "test_subscriptions_resource",
    "columns": [

      "2",
      "3"
    ]
  }

    "2",
    "3"
