POST /api/hosts/:host_id/puppetclass_ids
Puppet クラスをホストに追加

例

POST /api/hosts/host710/puppetclass_ids
{
  "puppetclass_id": 281110143,
  "host_class": {
    "puppetclass_id": 281110143
  }
}
200
{
  "host_id": 12,
  "puppetclass_id": 281110143
}

パラメーター

パラメーター名 記述
location_id
任意

Set the current location context for the request

Validations:

  • Must be a Integer
organization_id
任意

Set the current organization context for the request

Validations:

  • Must be a Integer
host_id
必須

ホストの ID

Validations:

  • Must be a String
puppetclass_id
必須

Puppet クラスの ID

Validations:

  • Must be a String
