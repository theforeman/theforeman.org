PUT /api/common_parameters/:id
Update a global parameter

Examples

PUT /api/common_parameters/636252244-test
{
  "common_parameter": {
    "name": "special_key",
    "value": "123"
  }
}
200
{
  "created_at": "2020-05-20 13:24:03 UTC",
  "updated_at": "2020-05-20 13:24:03 UTC",
  "hidden_value?": false,
  "hidden_value": "*****",
  "id": 636252244,
  "name": "special_key",
  "parameter_type": "string",
  "value": "123"
}
