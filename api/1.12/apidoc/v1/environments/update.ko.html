PUT /api/environments/:id
Update an environment.

Examples

PUT /api/environments/production
{
  "environment": {
    "name": "Development"
  }
}
200
{
  "environment": {
    "id": 334344675,
    "name": "Development",
    "created_at": "2016-06-15T12:53:26.000Z",
    "updated_at": "2016-06-15T12:53:32.070Z",
    "hosts_count": 0,
    "hostgroups_count": 0
  }
}

Params

Param Name Description
id
Required

Validations:

  • Must be an identifier, string from 1 to 128 characters containing only alphanumeric characters, space, underscore(_), hypen(-) with no leading or trailing space.
environment
Required

Validations:

  • Must be a Hash
environment[name]
Optional

Validations:

  • Must be String
