PUT /api/settings/:id
Update a setting.

Examples

PUT /api/settings/administrator
{
  "setting": {}
}
200
{
  "setting": {
    "id": 333972760,
    "name": "administrator",
    "value": "",
    "description": "The Default administrator email address",
    "settings_type": "string",
    "default": "root@some.host.fqdn",
    "created_at": "2016-06-15T12:53:26.000Z",
    "updated_at": "2016-06-15T12:53:42.191Z",
    "full_name": null
  }
}

Params

Param name Description
id
required

Validations:

  • Must be String
setting
required

Validations:

  • Must be a Hash
setting[value]
optional

Validations:
