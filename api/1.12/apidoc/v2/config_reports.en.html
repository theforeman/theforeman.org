GET /api/config_reports
List all reports

Examples

GET /api/hosts/host58/config_reports
200
{
  "total": 0,
  "subtotal": 0,
  "page": 1,
  "per_page": 20,
  "search": " host = host58",
  "sort": {
    "by": null,
    "order": null
  },
  "results": []
}

Params

Param name Description
search
optional

filter results

Validations:

  • Must be String
order
optional

sort results

Validations:

  • Must be String
page
optional

paginate results

Validations:

  • Must be String
per_page
optional

number of entries per request

Validations:

  • Must be String

Create a report

Param name Description

required

required

Validations:

  • Must be String

required

Validations:

  • Must be String

required

Hash of status type totals

Validations:

  • Must be Hash
config_report[metrics]
required

Validations:

  • Must be Hash

optional , nil allowed

Optional array of log hashes

  • Must be an array of any type
