POST /api/hosts/:host_id/interfaces
ホストにインターフェイスを作成

Examples

POST /api/hosts/host167/interfaces
{
  "interface": {
    "name": "test.foreman.com",
    "ip": "10.0.1.1",
    "mac": "AA:AA:AA:AA:AA:AA",
    "username": "foo",
    "password": "bar",
    "provider": "IPMI",
    "type": "UNKNOWN"
  }
}
422
{
  "error": {
    "message": "ERF13-6594 [InterfaceTypeMapper::UnknownTypeExeption]: Unknown interface type, must be one of [interface, bmc, bond, bridge]"
  }
}
