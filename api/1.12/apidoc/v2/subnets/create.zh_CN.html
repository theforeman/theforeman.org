Validations:

  • Must be String
subnet[network]
必填

  • Must be String

  • Must be String

  • Must be String

  • Must be String

选填 , 允许空值

这个子网的 IP 地址自动提示模式，有效值为 “DHCP”、“Internal DB”、“None”。

Validations:

  • Must be String

选填 , 允许空值

IP 自动提示的起始 IP 地址

Validations:

  • Must be String

选填 , 允许空值

Validations:

  • Must be String
subnet[vlanid]

Validations:

  • Must be String

Validations:

  • Must be a number.

Validations:

  • Must be a number.

选填 , 允许空值
