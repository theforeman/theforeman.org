参数

参数名字 描述
report
必填

Validations:

  • Must be a Hash
report[host]
必填

主机名或证书名称

Validations:

  • Must be String
report[reported_at]
必填

报告的 UTC 时间

Validations:

  • Must be String
report[status]
必填

状态类型总计哈希

Validations:

  • Must be Hash
report[metrics]
必填

报告指标散列，可只为 {}。

Validations:

  • Must be Hash
report[logs]
选填 , 允许空值

日志散列的自选阵列

Validations:

  • Must be an array of any type
