GET /api/media
Afficher tous les media d'installation

GET /api/operatingsystems/:operatingsystem_id/media
Afficher tous les media pour un système d'exploitation

GET /api/locations/:location_id/media
Afficher tous les media par emplacement

GET /api/organizations/:organization_id/media
Afficher tous les media par organisation
