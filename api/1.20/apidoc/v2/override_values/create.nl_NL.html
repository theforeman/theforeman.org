POST /api/smart_variables/:smart_variable_id/override_values
Maak een override waarde voor een specifieke smart variabele.

POST /api/smart_class_parameters/:smart_class_parameter_id/override_values
Maak een override waarde voor een specifieke smart class parameter

Examples

POST /api/smart_variables/1018350803/override_values
{
  "override_value": {
    "match": "domain=example.com",
    "value": "not_in_list"
  }
}
500
{
  "error": {
    "message": "Validation failed: Value not_in_list is not one of test, example, 30"
  }
}
