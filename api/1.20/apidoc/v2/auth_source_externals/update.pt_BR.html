PUT /api/auth_source_externals/:id
Update an external authentication source

Exemplos

PUT /api/auth_source_externals/408068537
{
  "auth_source_external": {
    "name": "new_name_external"
  }
}
200
{

  "id": 408068537,
  "name": "new_name_external",
  "external_usergroups": [],
  "locations": [],
  "organizations": []
}
