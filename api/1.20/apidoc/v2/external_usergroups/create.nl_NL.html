POST /api/usergroups/:usergroup_id/external_usergroups
Creeer een externe gebruikers groep gelinked aan een gebruikers groep

Examples

POST /api/usergroups/2-usergroup4/external_usergroups
{
  "external_usergroup": {
    "name": "foremanusergroup",
    "auth_source_id": 980190964
  }
}
201
{
  "id": 2,
  "name": "foremanusergroup",
  "auth_source_ldap": {
    "id": 980190964,
    "type": "AuthSourceLdap",
    "name": "auth_source_ldap_3"
  }
}
