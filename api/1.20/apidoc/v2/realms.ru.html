POST /api/realms
Создать область

The name field is used for the name of the realm.

Примеры

{
  "realm": {
    "name": "realm.net",
    "realm_proxy_id": "982965213-Realm Proxy",
    "realm_type": "FreeIPA"
  }
}

{

  "realm_proxy_id": 982965213,
  "realm_proxy_name": "Realm Proxy",
  "realm_proxy": {
    "name": "Realm Proxy",

    "url": "http://else.where:1337"
  },
  "locations": [],
  "organizations": []
}
