GET /api/smart_variables
列出所有智慧型變數

GET /api/hosts/:host_id/smart_variables
特定主機的智慧型變數之清單

GET /api/hostgroups/:hostgroup_id/smart_variables
特定主機群組的智慧型變數之清單

GET /api/puppetclasses/:puppetclass_id/smart_variables
特定 Puppet 類別的智慧型變數之清單
