POST /api/usergroups/:usergroup_id/external_usergroups
Criar um grupo externo ligado a um grupo de usuário

Exemplos

POST /api/usergroups/2-usergroup9/external_usergroups
{
  "external_usergroup": {
    "name": "foremanusergroup",
    "auth_source_id": "980190964"
  }
}
201
{
  "id": 2,
  "name": "foremanusergroup",
  "auth_source_ldap": {
    "id": 980190964,
    "type": "AuthSourceLdap",
    "name": "auth_source_ldap_4"
  }
}
