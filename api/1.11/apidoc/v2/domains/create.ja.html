POST /api/domains
ドメインの作成

The fullname field is used for human readability in reports and other pages that refer to domains, and also available as an external node parameter

Examples

POST /api/domains
{
  "domain": {
    "name": "domain.net"
  }
}
201
{
  "fullname": null,
  "dns_id": null,
  "created_at": "2016-03-10 15:51:54 UTC",
  "updated_at": "2016-03-10 15:51:54 UTC",
  "id": 759776764,
  "name": "domain.net",
  "subnets": [],
  "parameters": [],
  "interfaces": [],
  "locations": [],
  "organizations": []
}
