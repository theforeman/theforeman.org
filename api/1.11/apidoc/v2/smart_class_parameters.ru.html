GET /api/environments/:environment_id/smart_class_parameters
Список всех переопределяемых классов параметров для определенного окружения

Параметры

Имя параметра Описание
id
обязательно

Validations:

  • Must be an identifier, string from 1 to 128 characters containing only alphanumeric characters, space, underscore(_), hypen(-) with no leading or trailing space.
smart_class_parameter
обязательно

Validations:

  • Must be a Hash
smart_class_parameter[override]
необязательно

Значения переопределяемых классов параметров, управляемые Foreman

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[description]
необязательно

Описание переопределяемого класса

Validations:

  • Must be String
smart_class_parameter[default_value]
необязательно

Стандартное значение, если нет соответствий

Validations:

  • Must be String
smart_class_parameter[hidden_value]
необязательно

When enabled the parameter is hidden in the UI

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[use_puppet_default]
необязательно

Не передавайте этот параметр через ENC. Puppet будет использовать значение определенное в манифесте Puppet для этого параметра

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[path]
необязательно

Порядок обработки значений

Validations:

  • Must be String
smart_class_parameter[validator_type]
необязательно

Типы значений проверок

Validations:

  • Must be one of: regexp, list.
smart_class_parameter[validator_rule]
необязательно

Используется для обеспечения определенных значений для значений параметров

Validations:

  • Must be String
smart_class_parameter[override_value_order]
необязательно

Порядок обработки значений

Validations:

  • Must be String
smart_class_parameter[parameter_type]
необязательно

Типы значений переменных

Validations:

  • Must be one of: string, boolean, integer, real, array, hash, yaml, json.
smart_class_parameter[required]
необязательно

Если установлено, вызовет ошибку, если не установлено значение по умолчанию и не найдет поставщик значения

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[merge_overrides]
необязательно

Слить все совпадающие значения (только для типов массив/хэш)

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[merge_default]
необязательно

Include default value when merging all matching values

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[avoid_duplicates]
необязательно

Удалить задвоенные значения (только для массива)

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
