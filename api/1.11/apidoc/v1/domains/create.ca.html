POST /api/domains
Create a domain.

The fullname field is used for human readability in reports and other pages that refer to domains, and also available as an external node parameter

Examples

POST /api/domains
{
  "domain": {
    "name": "domain.net"
  }
}
201
{
  "domain": {
    "id": 759776764,
    "name": "domain.net",
    "fullname": null,
    "created_at": "2016-03-10T15:51:28.248Z",
    "updated_at": "2016-03-10T15:51:28.248Z",
    "dns_id": null,
    "total_hosts": 0,
    "hostgroups_count": 0
  }
}
