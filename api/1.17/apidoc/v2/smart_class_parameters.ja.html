  • Must be one of: true, false, 1, 0

  • Must be one of: true, false, 1, 0

任意

Foreman はこのパラメーターを分類出力で送信しません。Puppet はこのパラメーターについて Puppet マニフェストで定義された値を使用します

  • Must be one of: true, false, 1, 0
