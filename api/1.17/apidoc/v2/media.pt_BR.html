POST /api/media
Criar uma mídia

Parâmentros

Nome parâmetro Descrição
medium
requerido

Validations:

  • Must be a Hash
medium[name]
requerido

Nome da mídia

Validations:

  • Must be a String
medium[path]
requerido

The path to the medium, can be a URL or a valid NFS server (exclusive of the architecture).

for example mirror.centos.org/centos/$version/os/$arch where $arch will be substituted for the host's actual OS architecture and $version, $major and $minor will be substituted for the version of the operating system.

Solaris and Debian media may also use $release.

Validations:

  • Must be a String
medium[os_family]
opcional , nulo permitido

Sistema operacional Familia, os valores disponíveis :% { operatingsystem_families }

Validations:

  • Must be a String

opcional , nulo permitido

Validations:

  • Must be an array of any type

SUBSTITUIR localizações dadas com ids

Validations:
