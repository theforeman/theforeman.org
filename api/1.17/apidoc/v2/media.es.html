POST /api/media
Crear un medio

Parámetros

Nombre del parámetro Descripción
medium
requerido

Validations:

  • Must be a Hash
medium[name]
requerido

Nombre de medio

Validations:

  • Must be a String
medium[path]
requerido

The path to the medium, can be a URL or a valid NFS server (exclusive of the architecture).

for example mirror.centos.org/centos/$version/os/$arch where $arch will be substituted for the host's actual OS architecture and $version, $major and $minor will be substituted for the version of the operating system.

Solaris and Debian media may also use $release.

Validations:

  • Must be a String
medium[os_family]
opcional , nil permitido

Familia de sistema operativo, valores disponibles: AIX, Altlinux, Archlinux, Coreos, Debian, Freebsd, Gentoo, Junos, NXOS, Redhat, Solaris, Suse, Windows, Xenserver

Validations:

  • Must be a String
medium[operatingsystem_ids]
opcional , nil permitido

Validations:

  • Must be an array of any type
medium[location_ids]
opcional , nil permitido

REMPLAZAR ubicaciones con ID dadas

Validations:

  • Must be an array of any type
medium[organization_ids]
opcional , nil permitido

REMPLAZAR organizaciones con ID dadas.

Validations:

  • Must be an array of any type
