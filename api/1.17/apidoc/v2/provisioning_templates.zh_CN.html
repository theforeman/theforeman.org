  • Must be a String

Validations:

  • Must be an array of any type

选填 , 允许空值

与该模板关联的操作系统 ID 阵列

Validations:

  • Must be an array of any type
provisioning_template[locked]
选填 , 允许空值

Validations:

  • Must be one of: true, false, 1, 0

Validations:

  • Must be an array of any type
