POST /api/media
Créer un medium

Paramètres

Nom du paramètre Description
medium
requis

Validations:

  • Must be a Hash
medium[name]
requis

Nom du media

Validations:

  • Must be a String
medium[path]
requis

The path to the medium, can be a URL or a valid NFS server (exclusive of the architecture).

for example mirror.centos.org/centos/$version/os/$arch where $arch will be substituted for the host's actual OS architecture and $version, $major and $minor will be substituted for the version of the operating system.

Solaris and Debian media may also use $release.

Validations:

  • Must be a String
medium[os_family]
optionel , nil autorisé

Famille du système d'exploitation, valeurs disponibles : AIX, Altlinux, Archlinux, Coreos, Debian, Freebsd, Gentoo, Junos, NXOS, Redhat, Solaris, Suse, Windows, Xenserver

Validations:

  • Must be a String
medium[operatingsystem_ids]
optionel , nil autorisé

Validations:

  • Must be an array of any type
medium[location_ids]
optionel , nil autorisé

REMPLACER les emplacements qui ont les ID suivant

Validations:

  • Must be an array of any type
medium[organization_ids]
optionel , nil autorisé

REMPLACER les organisations qui ont les ID suivant

Validations:

  • Must be an array of any type
