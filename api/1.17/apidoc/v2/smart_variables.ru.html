POST /api/smart_variables

Параметры

Имя параметра Описание

обязательно

Validations:

  • Must be a Hash

обязательно

Имя переменной

Validations:

  • Must be a String
smart_variable[puppetclass_id]
необязательно , nil разрешен

Код класса Puppet

Validations:

  • Must be a number.
smart_variable[default_value]
необязательно , nil разрешен

Значение по умолчанию для переменной

Validations:

  • Must be one of: string, boolean, integer, real, array, hash, yaml, json.
smart_variable[hidden_value]
необязательно , nil разрешен

Если включено, параметр будет скрыт

Validations:

  • Must be one of: true, false, 1, 0
smart_variable[override_value_order]
необязательно , nil разрешен

Порядок обработки значений

Validations:

  • Must be a String
smart_variable[description]
необязательно , nil разрешен

Описание переменной

Validations:

  • Must be a String
smart_variable[validator_type]
необязательно , nil разрешен

Типы значений проверок

Validations:

  • Must be one of: regexp, list.
smart_variable[validator_rule]
необязательно , nil разрешен

Используется для обеспечения определенных значений для значений параметров

Validations:

  • Must be a String
smart_variable[variable_type]
необязательно , nil разрешен

Типы значений переменных

Validations:

  • Must be one of: string, boolean, integer, real, array, hash, yaml, json.
smart_variable[merge_overrides]
необязательно , nil разрешен

Слить все совпадающие значения (только для типов массив/хэш)

Validations:

  • Must be one of: true, false, 1, 0
smart_variable[merge_default]
необязательно , nil разрешен

При объединении совпадающих значений также включить значение, используемое по умолчанию

Validations:

  • Must be one of: true, false, 1, 0
smart_variable[avoid_duplicates]
необязательно , nil разрешен

Удалить повторяющиеся значения (только для массива)

Validations:

  • Must be one of: true, false, 1, 0
