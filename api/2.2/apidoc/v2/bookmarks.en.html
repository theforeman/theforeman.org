POST /api/bookmarks
Create a bookmark

Examples

POST /api/bookmarks
{
  "bookmark": {
    "public": false,
    "controller": "hosts",
    "name": "foo-bar",
    "query": "bar"
  }
}
201
{
  "name": "foo-bar",
  "controller": "hosts",
  "query": "bar",
  "public": false,
  "id": 980190963,
  "owner_id": 135138680,
  "owner_type": "User"
}
