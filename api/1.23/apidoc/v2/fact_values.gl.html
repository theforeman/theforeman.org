Fact values

>>>

GET /api/fact_values
List all fact values

GET /api/hosts/:host_id/facts
List all fact values of a given host

Examples

GET /api/hosts/host1933/facts
200
{
  "total": 1,
  "subtotal": 1,
  "page": 1,
  "per_page": 20,
  "search": " host = host1933",
  "sort": {
    "by": null,
    "order": null
  },
  "results": {
    "host1933": {
      "kernelversion": "2.6.9"
    }
  }
}
