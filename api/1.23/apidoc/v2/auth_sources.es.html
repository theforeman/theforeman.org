GET /api/auth_sources
Enumerar todas las fuentes de autenticación interna

GET /api/locations/:location_id/auth_sources
Enumerar las fuentes de autenticación interna por ubicación

GET /api/organizations/:organization_id/auth_sources
Enumerar las fuentes de autenticación por organización
