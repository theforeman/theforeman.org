POST /api/users/:user_id/table_preferences
为一个指定的表创建表偏好

示例

POST /api/users/135138680/table_preferences
{
  "name": "test_subscriptions_resource",
  "columns": [
    "1",
    "2",
    "3"
  ],
  "table_preference": {
    "name": "test_subscriptions_resource",
    "columns": [
      "1",
      "2",
      "3"
    ]
  }
}
201
{
  "id": 3,
  "name": "test_subscriptions_resource",
  "columns": [
    "1",
    "2",
    "3"
  ],
  "created_at": "2019-09-11 14:56:37 UTC",
  "updated_at": "2019-09-11 14:56:37 UTC"
}
