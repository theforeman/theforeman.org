POST /api/hosts/:host_id/puppetclass_ids
新增 Puppet 類別至主機

示例

POST /api/hosts/host2667/puppetclass_ids
{
  "puppetclass_id": "invalid_id",
  "host_class": {
    "puppetclass_id": "invalid_id"
  }
}
404
{
  "error": {
    "message": "Resource host_class not found by id ''"
  }
}
