GET /api/compute_resources/:id/storage_pods/:storage_pod_id
List attributes for a given storage pod

Examples

GET /api/compute_resources/305613588-Vmware/storage_pods/test_vmware_pod
200
{
  "name": "test_vmware_pod",
  "id": "group-p123456"
}
