POST /api/smart_variables
Create a smart variable.

Examples

POST /api/lookup_keys
{
  "lookup_key": {
    "is_param": true,
    "key": "testkey"
  }
}
201
{
  "lookup_key": {
    "required": false,
    "id": 980190963,
    "description": null,
    "key": "testkey",
    "is_param": true,
    "override": false,
    "default_value": null
  }
}
