PUT /api/domains/:id
Update a domain

Examples

PUT /api/domains/22495316-mydomain-net
{
  "domain": {
    "name": "domain.new"
  }
}
200
{
  "created_at": "2014-09-02T14:00:17Z",
  "dns_id": 113629430,
  "fullname": null,
  "hostgroups_count": 0,
  "hosts_count": 0,
  "id": 22495316,
  "name": "domain.new",
  "updated_at": "2014-09-02T14:00:38Z"
}

Params

Param name Description
id
required
Value: Must be an identifier, string from 1 to 128 characters containing only alphanumeric characters, space, underscore(_), hypen(-) with no leading or trailing space.
domain
required
Value: Must be a Hash
domain[name]
optional

The full DNS domain name


Value: Must be String
domain[fullname]
optional , nil allowed

Description of the domain


Value: Must be String
domain[dns_id]
optional , nil allowed

DNS proxy to use within this domain


Value: Must be a number.
domain[domain_parameters_attributes]
optional , nil allowed

Array of parameters (name, value)


Value: Must be an array of any type
