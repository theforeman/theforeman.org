>>>

POST /api/bookmarks
Create a bookmark.

Examples

POST /api/bookmarks
{
  "bookmark": {
    "public": false,
    "controller": "hosts",
    "name": "foo-bar",
    "query": "bar"
  }
}
200
{
  "name": "foo-bar",
  "controller": "hosts",
  "query": "bar",
  "public": false,
  "id": 980190965,
  "owner_id": 886836129,
  "owner_type": "User"
}

Params

Param name Description
bookmark
optional , nil allowed
Value: Must be a Hash
bookmark[name]
required
Value: Must be String
bookmark[controller]
required
Value: Must be String
bookmark[query]
required
Value: Must be String
bookmark[public]
optional , nil allowed
Value: Must be 'true' or 'false'
