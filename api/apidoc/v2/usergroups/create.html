POST /api/usergroups
Create a user group

Examples

POST /api/usergroups
{
  "usergroup": {
    "name": "test_usergroup"
  }
}
201
{
  "admin": false,
  "created_at": "2014-08-07T20:41:55Z",
  "id": 119,
  "name": "test_usergroup",
  "updated_at": "2014-08-07T20:41:55Z"
}
