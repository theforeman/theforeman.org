PUT /api/compute_resources/:id
Update a compute resource.

Examples

PUT /api/compute_resources/367690737-mycompute
{
  "compute_resource": {
    "description": "new_description"
  }
}
200
{
  "attrs": {},
  "created_at": "2014-07-23T13:30:50Z",
  "description": "new_description",
  "id": 367690737,
  "name": "mycompute",
  "password": "1234567",
  "updated_at": "2014-07-23T13:31:25Z",
  "url": "test:///default",
  "user": "MyString",
  "uuid": "mycompute"
}
