PUT /api/users/:id
Update an user.

Adds role ‘Anonymous’ to the user if it is not already present. Only admin can set admin account.

Examples

PUT /api/users/980190969
{
  "user": {
    "login": "johnsmith",
    "mail": "john@example.com",
    "auth_source_id": "200482051",
    "password": "123456"
  }
}
