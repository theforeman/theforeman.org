GET /api/hostgroups/:hostgroup_id/puppetclass_ids
List all Puppet class IDs for host group

Examples

GET /api/hostgroups/636252244/puppetclass_ids
200
{
  "results": [
    980190962
  ]
}
>>>

POST /api/hostgroups/:hostgroup_id/puppetclass_ids
Add a Puppet class to host group

Examples

POST /api/hostgroups/636252244/puppetclass_ids
{
  "puppetclass_id": "281110143",
  "hostgroup_class": {
    "puppetclass_id": "281110143"
  }
}
200
{
  "hostgroup_id": 636252244,
  "puppetclass_id": 281110143
}

Params

Param name Description
hostgroup_id
required

ID of host group


Value: Must be String
puppetclass_id
required

ID of Puppet class


Value: Must be String
