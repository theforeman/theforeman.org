GET /api/usergroups/:usergroup_id/external_usergroups/:id
Show an external user group for user group

Examples

GET /api/usergroups/110/external_usergroups/8
200
{
  "id": 8,
  "name": "external_usergroup2",
  "auth_source_ldap": {
    "id": 980190966,
    "type": "AuthSourceLdap",
    "name": "auth_source_ldap_2"
  }
}

Params

Param name Description
usergroup_id
required

ID or name of user group


Value: Must be String
id
required

ID or name of external user group


Value: Must be String
