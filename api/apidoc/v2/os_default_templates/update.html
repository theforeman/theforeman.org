PUT /api/operatingsystems/:operatingsystem_id/os_default_templates/:id
Update a os default template for operating system

Examples

PUT /api/operatingsystems/1073012828/os_default_templates/980190962
{
  "os_default_template": {
    "config_template_id": "821548108",
    "template_kind_id": "452984334"
  }
}
200
{
  "id": 980190962,
  "config_template_id": 821548108,
  "config_template_name": "PXELinux default local boot",
  "template_kind_id": 452984334,
  "template_kind_name": "PXELinux"
}

Params

Param name Description
operatingsystem_id
optional

id of operating system


Value: Must be String

Value: Must be String

Value: Must be a Hash
