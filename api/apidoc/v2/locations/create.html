POST /api/locations
Create a location

Examples

POST /api/locations
{
  "location": {
    "name": ""
  }
}
500
{
  "error": {
    "message": "undefined method `permission_failed?' for #<Location:0x0000000f427480>"
  }
}

Params

Param name Description
location
optional , nil allowed
Value: Must be a Hash
location[name]
required
Value: Must be String
