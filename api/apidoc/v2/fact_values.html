Fact values

>>>

GET /api/fact_values
List all fact values.

GET /api/hosts/:host_id/facts
List all fact values of a given host.

Examples

GET /api/fact_values
200
{
  "total": 2,
  "subtotal": 2,
  "page": 1,
  "per_page": 20,
  "search": "",
  "sort": {
    "by": null,
    "order": null
  },
  "results": {
    "my5name.mydomain.net": {
      "ipaddress": "10.0.19.33",
      "kernelversion": "2.6.9"
    }
  }
}

Params

Param name Description
search
optional

filter results


Value: Must be String
order
optional

sort results


Value: Must be String
page
optional

paginate results


Value: Must be String
per_page
optional

number of entries per request


Value: Must be String
