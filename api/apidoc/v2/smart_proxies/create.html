POST /api/smart_proxies
Create a smart proxy.

Examples

POST /api/smart_proxies
{
  "smart_proxy": {
    "name": "master02",
    "url": "http://server:8443"
  }
}
201
{
  "created_at": "2014-07-23T13:32:10Z",
  "id": 982965215,
  "name": "master02",
  "updated_at": "2014-07-23T13:32:10Z",
  "url": "http://server:8443"
}

Params

Param name Description
smart_proxy
optional , nil allowed
Value: Must be a Hash
smart_proxy[name]
required
Value: Must be String
smart_proxy[url]
required
Value: Must be String
