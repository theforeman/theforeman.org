Param name Description

optional
Value: Must be String
environment_id
optional
Value: Must be String
dryrun
optional
Value: Must be 'true' or 'false'
except
optional

Optional comma-deliminated string containing either ‘new,updated,obsolete’ used to limit the import_puppetclasses actions


Value: Must be String
