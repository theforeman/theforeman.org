POST /api/smart_variables/:smart_variable_id/override_values
Create an override value for a specific smart_variable

POST /api/smart_class_parameters/:smart_class_parameter_id/override_values
Create an override value for a specific smart class parameter

Examples

POST /api/smart_variables/281110143-special_info/override_values
{
  "override_value": {
    "match": "xyz=10",
    "value": "string"
  }
}
200
{
  "id": 1018350796,
  "match": "xyz=10",
  "value": "string",
  "created_at": "2014-07-23T13:31:58Z",
  "updated_at": "2014-07-23T13:31:58Z"
}

Params

Param name Description
smart_variable_id
optional
Value: Must be an identifier, string from 1 to 128 characters containing only alphanumeric characters, space, underscore(_), hypen(-) with no leading or trailing space.
override_value
optional , nil allowed
Value: Must be a Hash
override_value[match]
optional , nil allowed
Value: Must be String
override_value[value]
optional , nil allowed
Value: Must be String
