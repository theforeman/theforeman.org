POST /api/hosts/:host_id/puppetclass_ids
Add a puppetclass to host

Examples

POST /api/hosts/my5name.mydomain.net/puppetclass_ids
{
  "puppetclass_id": "281110143"
}
200
{
  "host_id": 980190962,
  "puppetclass_id": 281110143
}

Params

Param name Description
host_id
required

id of host


Value: Must be String
puppetclass_id
required

id of puppetclass


Value: Must be String
