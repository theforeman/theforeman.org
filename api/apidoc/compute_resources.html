POST /api/compute_resources
Create a compute resource.

Examples

POST /api/compute_resources
{
  "compute_resource": {
    "name": "special_compute",
    "user": "user@example.com",
    "provider": "EC2",
    "password": "secret",
    "region": "eu-west-1"
  }
}
200
{
  "compute_resource": {
    "name": "special_compute",
    "url": "eu-west-1",
    "user": "user@example.com",
    "provider": "EC2",
    "id": 980190963,
    "description": null,
    "updated_at": "2012-12-18T15:24:44Z",
    "created_at": "2012-12-18T15:24:44Z"
  }
}

Value: Must be String

Value: Must be String
