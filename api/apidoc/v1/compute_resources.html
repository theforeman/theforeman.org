POST /api/compute_resources
Create a compute resource.

Examples

POST /api/compute_resources
{
  "compute_resource": {
    "name": "special_compute",
    "region": "eu-west-1",
    "user": "user@example.com",
    "password": "secret"
  }
}
200
{
  "compute_resource": {
    "id": 980190963,
    "name": "special_compute",
    "description": null,
    "url": "eu-west-1",
    "created_at": "2014-08-07T20:40:16Z",
    "updated_at": "2014-08-07T20:40:16Z",
    "provider": "EC2",
    "user": "user@example.com",
    "region": "eu-west-1"
  }
}

Param name Description
compute_resource
required
Value: Must be a Hash
compute_resource[name]
required
Value: Must be String

optional

Value: Must be String

required

URL for Libvirt, Ovirt, and Openstack


Value: Must be String

optional
Value: Must be String

optional

optional

Value: Must be String

Value: Must be String
