POST /api/architectures
Create an architecture.

Examples

POST /api/architectures
{
  "architecture": {
    "name": "i386"
  }
}
200
{
  "architecture": {
    "name": "i386",
    "id": 501905020,
    "created_at": "2014-08-07T20:40:13Z",
    "operatingsystem_ids": [],
    "updated_at": "2014-08-07T20:40:13Z"
  }
}

Params

Param name Description
architecture
required
Value: Must be a Hash
architecture[name]
required
Value: Must be String
architecture[operatingsystem_ids]
optional

Operatingsystem ID’s


Value: Must be Array
