GET /api/hosts/:id/status
Get status of host

Return value may either be one of the following:

  • missing
  • failed
  • pending
  • changed
  • unchanged
  • unreported

Examples

GET /api/hosts/my5name.mydomain.net/status
200
{
  "status": "No reports"
}
