PUT /api/ptables/:id
Update a ptable.

Examples

PUT /api/ptables/980190962
{
  "ptable": {}
}
200
{
  "ptable": {
    "created_at": "2014-08-07T20:40:10Z",
    "id": 980190962,
    "layout": "zerombr yes\\nclearpart --all    --initlabel\\npart /boot --fstype ext3 --size=100 --asprimary\\npart /     --f   stype ext3 --size=1024 --grow\\npart swap  --recommended",
    "name": "default",
    "os_family": null,
    "updated_at": "2014-08-07T20:40:10Z"
  }
}

Params

Param name Description
id
required

ptable

Value: Must be a Hash
ptable[name]
