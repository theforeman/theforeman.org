POST /api/environments
Create an environment.

Examples

POST /api/environments
{
  "environment": {
    "name": "Development"
  }
}
200
{
  "environment": {
    "name": "Development",
    "id": 687036938,
    "created_at": "2014-07-23T13:31:00Z",
    "updated_at": "2014-07-23T13:31:00Z"
  }
}

Params

Param name Description
environment
required
Value: Must be a Hash
environment[name]
required
Value: Must be String
