POST /api/domains
Create a domain.

The fullname field is used for human readability in reports and other pages that refer to domains, and also available as an external node parameter

Examples

POST /api/domains
{
  "domain": {
    "name": "domain.net"
  }
}
201
{
  "domain": {
    "created_at": "2014-08-07T20:40:20Z",
    "dns_id": null,
    "fullname": null,
    "hostgroups_count": 0,
    "hosts_count": 0,
    "id": 759776764,
    "name": "domain.net",
    "updated_at": "2014-08-07T20:40:20Z"
  }
}
