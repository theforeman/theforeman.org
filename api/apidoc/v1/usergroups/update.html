PUT /api/usergroups/:id
Update a usergroup.

Examples

PUT /api/usergroups/107
{
  "usergroup": {}
}
200
{
  "usergroup": {
    "admin": false,
    "created_at": "2014-08-07T20:40:44Z",
    "id": 107,
    "name": "usergroup4",
    "updated_at": "2014-08-07T20:40:44Z"
  }
}

Params

Param name Description
id
required
Value: Must be String
usergroup
required
Value: Must be a Hash
usergroup[name]
optional
Value: Must be String
