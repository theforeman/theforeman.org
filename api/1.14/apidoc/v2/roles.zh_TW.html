POST /api/roles
建立角色

示例

POST /api/roles
{
  "role": {
    "name": "staff"
  }
}
201
{
  "builtin": 0,
  "name": "staff",
  "id": 13,
  "description": null,
  "filters": [],
  "locations": [],
  "organizations": []
}

參數

參數名字 描述
role
必填

Validations:

  • Must be a Hash
role[name]
必填

Validations:

  • Must be String
