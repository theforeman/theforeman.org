GET /api/hosts/:id/template/:kind
렌더링된 프로비저닝 템플릿 컨텐츠 미리 보기

Examples

GET /api/hosts/host109.example16.com/template/provitamin
404
{
  "message": "No template with kind provitamin for host109.example16.com"
}

Params

Param Name Description
id
Required

Validations:

  • Must be an identifier, string from 1 to 128 characters containing only alphanumeric characters, dot(.), space, underscore(_), hypen(-) with no leading or trailing space.
kind
Required

템플릿 종류로, 사용 가능한 값은 다음과 같습니다: PXELinux, iPXE, provision, finish, script, PXEGrub, user_data, ZTP, Bootdisk, POAP, PXEGrub2

Validations:

  • Must be String
