GET /api/users/:user_id/personal_access_tokens/:id
Show a Personal Access Token for a user

Examples

GET /api/users/988725685/personal_access_tokens/7
200
{
  "created_at": "2019-02-20 13:34:48 UTC",
  "updated_at": "2019-02-20 13:34:48 UTC",
  "id": 7,
  "name": "Personal Access Token 11",
  "expires_at": "2019-03-02 13:34:48 UTC",
  "active?": true,
  "user_id": 988725685,
  "last_used_at": null
}

Params

Param Name Description
location_id
Optional

organization_id
Optional

Scope by organizations

Validations:

id
Required

Validations:

Required

ID of the user

Validations:
