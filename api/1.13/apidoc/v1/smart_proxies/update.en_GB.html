PUT /api/smart_proxies/:id
Update a smart proxy.

Examples

PUT /api/smart_proxies/980190962-DHCP%20Proxy
{
  "smart_proxy": {
    "name": "master02",
    "url": "http://server:8443"
  }
}
200
{
  "smart_proxy": {
    "id": 980190962,
    "name": "master02",
    "url": "http://server:8443",
    "created_at": "2016-09-30T07:53:50.000Z",
    "updated_at": "2016-09-30T07:54:22.113Z",
    "expired_logs": "0"
  }
}
