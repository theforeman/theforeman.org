POST /api/config_templates
Create a template

示例

POST /api/config_templates
{
  "config_template": {
    "name": ""
  }
}
422
{
  "provisioning_template": {
    "id": null,
    "errors": {
      "name": [
        "can't be blank"
      ],
      "template": [
        "can't be blank"
      ],
      "template_kind_id": [
        "can't be blank"
      ]
    },
    "full_messages": [
      "Name can't be blank",
      "Template can't be blank",
      "Template kind can't be blank"
    ]
  }
}

参数

参数名字 描述
config_template
必填

Validations:

  • Must be a Hash
config_template[name]
必填

模板名称

Validations:

  • Must be String
config_template[template]
必填

Validations:

  • Must be String
config_template[snippet]
选填 , 允许空值

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
config_template[audit_comment]
选填 , 允许空值

Validations:

  • Must be String
config_template[template_kind_id]
选填 , 允许空值

与片段无关

Validations:

  • Must be a number.
config_template[template_combinations_attributes]
选填

模板组合阵列（hostgroup_id, environment_id）

Validations:

  • Must be an array of any type
config_template[operatingsystem_ids]
选填

Array of operating systems ID to associate the template with

Validations:

  • Must be an array of any type
