PUT /api/usergroups/:id
Update a user group.

Examples

PUT /api/usergroups/1-usergroup3
{
  "usergroup": {
    "name": "test_usergroup"
  }
}
200
{
  "usergroup": {
    "id": 1,
    "name": "test_usergroup",
    "created_at": "2016-09-30T07:54:23.434Z",
    "updated_at": "2016-09-30T07:54:23.456Z",
    "admin": false
  }
}

Params

Param Name Description
id
Required

Validations:

  • Must be String
usergroup
Required

Validations:

  • Must be a Hash
usergroup[name]
Optional

Validations:

  • Must be String
