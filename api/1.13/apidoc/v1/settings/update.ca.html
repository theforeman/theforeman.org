PUT /api/settings/:id
Update a setting.

Examples

PUT /api/settings/231989830
{
  "setting": {
    "value": "100"
  }
}
200
{
  "setting": {
    "id": 231989830,
    "name": "outofsync_interval",
    "value": 100,
    "description": "Duration in minutes after the Puppet interval for servers to be classed as out of sync.",
    "settings_type": "integer",
    "default": 5,
    "created_at": "2016-09-30T07:53:50.000Z",
    "updated_at": "2016-09-30T07:54:18.616Z",
    "full_name": null
  }
}

Params

Param Name Description
id
Required

Validations:

  • Must be String
setting
Required

Validations:

  • Must be a Hash
setting[value]
Optional

Validations:

  • Must be String
