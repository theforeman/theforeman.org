Smart class parameters

>>>

GET /api/smart_class_parameters
Alle Smart-Klassenparameter auflisten

GET /api/hosts/:host_id/smart_class_parameters
Liste der Smart-Klassenparameter für einen bestimmten Host

GET /api/hostgroups/:hostgroup_id/smart_class_parameters
Liste der Smart-Klassenparameter für eine bestimmte Hostgruppe

GET /api/puppetclasses/:puppetclass_id/smart_class_parameters
Liste der Smart-Klassenparameter für eine bestimmte Puppetklasse

GET /api/environments/:environment_id/smart_class_parameters
Liste der Smart-Klassenparameter für eine bestimmte Umgebung

GET /api/environments/:environment_id/puppetclasses/:puppetclass_id/smart_class_parameters
Liste der Smart-Klassenparameter für eine bestimmte Umgebung/Puppet-Klasse-Kombination

Parameter

Parametername Beschreibung
id
verpflichtend

Validations:

  • Must be an identifier, string from 1 to 128 characters containing only alphanumeric characters, space, underscore(_), hypen(-) with no leading or trailing space.
smart_class_parameter
verpflichtend

Validations:

  • Must be a Hash
smart_class_parameter[override]
optional

Ob der Smart-Klassenparameter von Foreman gemanagt wird

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[description]
optional

Beschreibung der Smart-Klasse

Validations:

  • Must be String
smart_class_parameter[default_value]
optional

Zu verwendender Wert, falls es keine Übereinstimmung gibt

Validations:

  • Must be String
smart_class_parameter[hidden_value]
optional

Wenn aktiviert, so ist der Parameter im UI verborgen

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[use_puppet_default]
optional

Übermitteln Sie diesen Parameter nicht über den ENC. Puppet wird den im Puppetmanifest für diesen Parameter definierten Wert verwenden.

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[path]
optional

Reihenfolge, in der Werte aufgelöst werden

Validations:

  • Must be String
smart_class_parameter[validator_type]
optional

Typen von Validierungswerten

Validations:

  • Must be one of: regexp, list.
smart_class_parameter[validator_rule]
optional

Verwendet, um bestimmte Werte für die Parameterwerte umzusetzen

Validations:

  • Must be String
smart_class_parameter[override_value_order]
optional

Reihenfolge, in der Werte aufgelöst werden

Validations:

  • Must be String
smart_class_parameter[parameter_type]
optional

Typen von Variablenwerten

Validations:

  • Must be one of: string, boolean, integer, real, array, hash, yaml, json.
smart_class_parameter[required]
optional

Falls “True”, wird dies einen Fehler erzeugen, wenn es keinen Standardwert gibt und kein Treffer einen Wert liefert

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[merge_overrides]
optional

Alle übereinstimmenden Werte (nur vom Typ Array/Hash) zusammenführen

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[merge_default]
optional

Standardwert einschließen, wenn alle übereinstimmenden Werte zusammengeführt werden

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
smart_class_parameter[avoid_duplicates]
optional

Duplikate (nur vom Typ Array) entfernen

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
