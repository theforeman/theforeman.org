POST /api/realms
创建范围

The name field is used for the name of the realm.

示例

POST /api/realms
{
  "realm": {
    "name": "realm.net",
    "realm_proxy_id": "982965213-Realm Proxy",
    "realm_type": "FreeIPA"
  }
}
201
{
  "name": "realm.net",
  "realm_proxy_id": 982965213,
  "realm_type": "FreeIPA",
  "created_at": "2016-09-30 07:55:09 UTC",
  "updated_at": "2016-09-30 07:55:09 UTC",
  "id": 936160440,
  "locations": [],
  "organizations": []
}

参数

参数名字 描述
realm
必填

Validations:

  • Must be a Hash
realm[name]
必填

范围名称，例如：EXAMPLE.COM

Validations:

  • Must be String
realm[realm_proxy_id]
必填 , 允许空值

这个范围中使用的代理服务器

Validations:

  • Must be a number.
realm[realm_type]
必填

范围类型，例如：FreeIPA 或 Active Directory

Validations:

  • Must be String
