PUT /api/smart_variables/:smart_variable_id/override_values/:id
Обновить переопределение смарт-переменной

PUT /api/smart_class_parameters/:smart_class_parameter_id/override_values/:id
Обновить значение переопределения смарт-параметра

Примеры

PUT /api/smart_class_parameters/328495940-cluster/override_values/874520030
{
  "override_value": {
    "match": "host=abc.com"
  }
}
200
{
  "created_at": "2016-09-30 07:53:50 UTC",
  "updated_at": "2016-09-30 07:55:02 UTC",
  "id": 874520030,
  "match": "host=abc.com",
  "value": "8080",
  "use_puppet_default": false
}

Параметры

Имя параметра Описание
smart_variable_id
необязательно

Validations:

  • Must be an identifier, string from 1 to 128 characters containing only alphanumeric characters, space, underscore(_), hypen(-) with no leading or trailing space.
smart_class_parameter_id
необязательно

Validations:

  • Must be an identifier, string from 1 to 128 characters containing only alphanumeric characters, space, underscore(_), hypen(-) with no leading or trailing space.
override_value
обязательно

Validations:

  • Must be a Hash
override_value[match]
необязательно
