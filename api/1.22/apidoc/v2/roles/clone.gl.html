POST /api/roles/:id/clone
Clone a role

Examples

POST /api/roles/13-Organization%20admin/clone
{
  "new_name": "Org Admin",
  "role": {
    "name": "Org Admin"
  }
}
201
{
  "id": 18,
  "name": "Org Admin",
  "builtin": 0,
  "description": null,
  "origin": null,
  "cloned_from_id": 13
}
