PUT /api/common_parameters/:id
Update a global parameter

Examples

PUT /api/common_parameters/636252244-test
{
  "common_parameter": {
    "name": "special_key",
    "value": "123"
  }
}
200
{
  "created_at": "2019-02-20 13:35:00 UTC",
  "updated_at": "2019-02-20 13:35:08 UTC",
  "hidden_value?": false,
  "hidden_value": "*****",
  "id": 636252244,
  "name": "special_key",
  "value": "123"
}
