GET /api/fact_values
すべてのファクト値を一覧表示

GET /api/hosts/:host_id/facts
指定されたホストのすべてのファクト値を一覧表示

例

GET /api/hosts/host5/facts
200
{
  "total": 1,
  "subtotal": 1,
  "page": 1,
  "per_page": 20,
  "search": " host = host5",
  "sort": {
    "by": null,
    "order": null
  },
  "results": {
    "host5": {
      "kernelversion": "2.6.9"
    }
  }
}
