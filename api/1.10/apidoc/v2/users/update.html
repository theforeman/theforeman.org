PUT /api/users/:id
Update a user

Adds role ‘Anonymous’ to the user if it is not already present. Only another admin can change the admin account attribute.

Examples

PUT /api/users/980190963
{
  "user": {
    "login": "johnsmith",
    "mail": "john@example.com",
    "auth_source_id": "200482051",
    "password": "123456"
  }
}
