POST /api/roles
Create a role

Examples

POST /api/roles
{
  "role": {
    "name": "staff"
  }
}
201
{
  "builtin": 0,
  "name": "staff",
  "id": 13,
  "filters": []
}

Params

Param name Description
role
required
Value: Must be a Hash
role[name]
required
Value: Must be String
