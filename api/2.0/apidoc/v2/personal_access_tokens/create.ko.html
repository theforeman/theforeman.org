POST /api/users/:user_id/personal_access_tokens
Create a Personal Access Token for a user

Examples

POST /api/users/988725682/personal_access_tokens
{
  "personal_access_token": {
    "name": "foreman@example.com",
    "expires_at": "2020-04-08T11:19:39Z"
  }
}

  "id": 3,

  "expires_at": "2020-04-08 11:19:39 UTC",
  "active?": true,

  "last_used_at": null
}
