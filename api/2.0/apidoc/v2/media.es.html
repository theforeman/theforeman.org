Listar todos los medios de instalación

Listar todos los medios para un sistema operativo

GET /api/locations/:location_id/media
Listar todos los medios por ubicación

GET /api/organizations/:organization_id/media
Listar todos los medios por organización

Alcance por organización

Validations:

opcional

filtrar resultados

Validations:

POST /api/media
Crear un medio

Parámetros

Nombre del parámetro Descripción
location_id
opcional

Set the current location context for the request

Validations:

  • Must be a Integer
organization_id
opcional

Set the current organization context for the request

Validations:

  • Must be a Integer
medium
requerido

Validations:

  • Must be a Hash
medium[name]
requerido

Nombre de medio

Validations:

  • Must be a String
medium[path]
requerido

The path to the medium, can be a URL or a valid NFS server (exclusive of the architecture).

for example http://mirror.centos.org/centos/$version/os/$arch where $arch will be substituted for the host’s actual OS architecture and $version, $major and $minor will be substituted for the version of the operating system.

Solaris and Debian media may also use $release.

Validations:

  • Must be a String
medium[os_family]
opcional , nil permitido

Familia de sistema operativo, valores disponibles: AIX, Altlinux, Archlinux, Coreos, Debian, Freebsd, Gentoo, Junos, NXOS, Rancheros, Redhat, Solaris, Suse, VRP, Windows, Xenserver

Validations:

  • Must be a String
medium[operatingsystem_ids]
opcional , nil permitido

Validations:

  • Must be an array of any type
medium[location_ids]
opcional , nil permitido

REMPLAZAR ubicaciones con ID dadas

Validations:

  • Must be an array of any type
medium[organization_ids]
opcional , nil permitido

REMPLAZAR organizaciones con ID dadas.

Validations:

  • Must be an array of any type
