GET /api/fact_values
Listar todos los valores de eventos

GET /api/hosts/:host_id/facts
Listar todos los valores de eventos de un host determinado

Ejemplos

GET /api/fact_values
200
{
  "total": 1,
  "subtotal": 1,
  "page": 1,
  "per_page": 20,

  "sort": {
    "by": null,
    "order": null
  },
  "results": {
    "host91": {
      "kernelversion": "2.6.9"
    }
  }
}
