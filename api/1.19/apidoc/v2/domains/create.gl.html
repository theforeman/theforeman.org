POST /api/domains
Create a domain

The fullname field is used for human readability in reports and other pages that refer to domains, and also available as an external node parameter

Examples

POST /api/domains
{
  "organization_id": null,
  "domain": {
    "name": "example207.com"
  }
}
201
{
  "fullname": null,
  "created_at": "2018-11-08 09:07:30 UTC",
  "updated_at": "2018-11-08 09:07:30 UTC",
  "id": 759776764,
  "name": "example207.com",
  "dns_id": null,
  "dns": null,
  "subnets": [],
  "interfaces": [],
  "parameters": [],
  "locations": [],
  "organizations": []
}
