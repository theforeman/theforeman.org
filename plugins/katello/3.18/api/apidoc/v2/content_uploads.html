POST /katello/api/repositories/:repository_id/content_uploads
Create an upload request

Params

Param name Description
repository_id
required

repository id

Validations:

  • Must be a number.
size
required

Size of file to upload

Validations:

  • Must be a number.
checksum
optional

Checksum of file to upload

Validations:

  • String
content_type
optional

content type (‘deb’, ‘docker_manifest’, ‘file’, ‘ostree’, ‘puppet_module’, ‘rpm’, ‘srpm’)

Validations:

  • Must be one of: deb, docker_manifest, file, ostree, puppet_module, rpm, srpm.
